# 🌐 Proyecto Web - Página en HTML y CSS

## 📌 Descripción

Este proyecto consiste en el desarrollo de una página web como parte de una tarea académica. La página ha sido creada utilizando **HTML y CSS**, integrando diferentes elementos fundamentales del desarrollo web.

## 🎯 Objetivo

El objetivo principal es aplicar los conocimientos básicos de maquetación y diseño web, incluyendo la estructura de páginas y estilos visuales.

## 🧩 Contenido del proyecto

La página web incluye:

* 🖼️ Imágenes
* 🔘 Botones interactivos
* 📊 Tablas
* 📝 Formularios
* 🎨 Estilos con CSS

## Tecnologías utilizadas

* HTML5
* CSS3
